# Reason IDEA Plugin

[**reasonml-editor.github.io/reasonml-idea-plugin**](https://reasonml-editor.github.io/reasonml-idea-plugin/)

ReasonML language plugin for IDEA. Supports Reason, ReScript and OCaml.
